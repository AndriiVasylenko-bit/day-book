# DayBook

### Task

* Описати клас з іменем STUDENT, який містить наступні поля: 
	* прізвище;
	* номер групи;
	* успішність(масив із п’яти елементів).

* Написати програму, яка виконує наступні дії:
	* зчитування даних з рядкового потоку в масив, який складається структур типу STUDENT.
	* впорядкувати їх за зростанням номера групи; 
	* виведення на дисплей прізвищ і номерів груп для всіх студентів, включених до масиву, якщо середній бал студента більший ніж 4.0; якщо таких студентів немає, вивести відповідне повідомлення.
	* реалізувати пошук інформації по не повному прізвищу
